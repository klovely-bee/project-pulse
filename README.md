# Project Pulse

Project Pulse is a fullstack application for managing weekly activity reports, peer evaluations, and senior design administration for TCU senior design courses.

## Repository Structure

```text
project-pulse/
├── backend/   # Spring Boot API
├── frontend/  # Vue 3 application
├── docs/      # Project planning and branch mapping
└── .idea/project-pulse-ai/  # Reference requirements and design docs only
```

## Reference Material

Implementation in this repository follows the requirements and design guidance stored in `.idea/project-pulse-ai/`.

- Use cases: `.idea/project-pulse-ai/requirements/use_cases.md`
- Architecture: `.idea/project-pulse-ai/docs/architecture.md`
